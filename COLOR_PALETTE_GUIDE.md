## 🎨 HOW TO USE IN CODE

### Using Colors in Tailwind Classes:
```jsx
<div className="bg-electric-cyan/10 border-royal-purple text-amber-gold">
  Your content here
</div>
```

### Using Gradient Text:
```jsx
<h1 className="gradient-text-rainbow">Rainbow Title</h1>
<h2 className="gradient-text-ocean">Ocean Title</h2>
<p className="gradient-text-cosmic">Cosmic Text</p>
```

### Using Gradient Backgrounds:
```jsx
<div className="bg-gradient-sunset p-6">
  Sunset themed card
</div>
```

### Using Color Variables in Inline Styles:
```jsx
<div style={{ borderTopColor: 'var(--color-electric-cyan)' }}>
  Custom colored element
</div>
```
